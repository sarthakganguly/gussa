Product Requirements Document: The Sovereign Vault
==================================================

1\. Executive Summary
---------------------

**Gussa (also known as the Sovereign Vault)** is a privacy-first, offline-only mental health sanctuary designed for the "Digital Hermit." Unlike market leaders that rely on cloud-sync and engagement metrics, this application prioritizes **Zero-Cloud Sovereignty** and **Time to Relief**.

### Core Philosophy

-   **Privacy by Design:** No accounts, no servers, no data collection. The data belongs to the device.

-   **Reliability:** Functions perfectly in "Dead Zones" (Airplanes, remote areas, subways).

-   **Anti-App HUD:** A minimalist "Heads-Up Display" designed to get the user in, supported, and back to their life quickly.

2\. Unique Value Propositions (UVP)
-----------------------------------

1.  **Zero-Cloud Sovereignty:** Absolute data privacy; the app never connects to a backend.

2.  **On-Device AI Coaching:** Empathetic CBT/DBT support powered by local hardware (Gemini Nano via Android AICore).

3.  **Perpetual Ownership:** A one-time purchase model (no subscriptions).

4.  **Interoperable Sovereignty:** Users can export data at any time in open formats (JSON/Markdown).

5.  **Offline Reliability:** 100% of the library and tools are stored locally.

3\. Visual Identity & UX
------------------------

-   **Vibe:** "Grounded Optimism" --- warm, soothing, tactile.

-   **Color Palette:** Soft Parchment (`#FDF9F3`), Deep Walnut (`#3D3633`), Morning Sage (`#A3B18A`), Terracotta Dust (`#D4A373`).

-   **Typography:** Primary: Manrope (Sans-Serif, variable weights). Secondary: Figtree.

-   **Design Language:** "Soft Geometry" (16px-24px radii), minimal shadows, flat paper-like borders, tactile haptic feedback.

4\. Feature Roadmap
-------------------

-   **Phase 1 (Android MVP):** Emergency SOS Toolkit, Icon-Based Micro-Journaling, Private AI Coach (Gemini Nano), Clinical Baseline (PHQ-9/GAD-7), Daily Mental Health Card.

-   **Phase 2 (Retention):** Procedural Soundscapes, Gamification (Streaks), Pattern Discovery, Local Notifications.

-   **Phase 3 (iOS & Content):** iOS Port via Capacitor, Audio Content Library, Structured Packs.

-   **Phase 4 (Sovereignty 2.0):** Peer-to-Peer (P2P) Sync, Biometric Vault Lock.

5\. Product Constraints (Solo Engineer Focus)
---------------------------------------------

1.  **No-Backend:** The app must function without a database server or API gateway.

2.  **Content-Light MVP:** Prioritize logic-based tools over massive media production.

3.  **Local Encryption:** All journaling and AI data must be encrypted at rest using device-level keys.
